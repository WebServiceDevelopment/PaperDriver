1. Prep Stage (事務所で行う) Reset

1. 運行指示書

「運行指示書」の元となるデータ(.csv/.xlsx) をネットワークストレージから選択する

2. 特殊車両通行許可申請書

「特殊車両通行許可申請書」のPDFファイルを ネットワークストレージから選択する

3. 運行標準書

「運行標準書」の元となるデータ(.csv/.xlsx) をネットワークストレージから選択する

2. Deliver Stage (トラックで行う)

4. 点検

「点検」タブからを入力する 点検入力した後、「点検確定」ボタンを押す

5. 開始メーター

「運転記録」タブから「開始オドメーター」と 「出庫時間」を入力した後、「開始確認」を押す

6. 終了メーター

「運転記録」タブから「 終了オドメーター」と 「退庫時間」を入力した後、「終了確認」を押す

3. Return Stage (事務所で行う)

7. デジタコデータ読込

ネットワークストレージから デジタコのCSVデータを選択する

8. 日報調整

「運転日報」のページは調整のために編集可能
編集完了した際は「日報確定」を押す

9. Export PDF's

本件のデータ（CSVやPDF）を書き出す
保管するには「運転記録」タブから「データ書き出し」を押す
